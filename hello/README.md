### Simple implementation of [Hello World](https://en.wikipedia.org/wiki/%22Hello,_World!%22_program) in Juooga.

#### Emojis 😜

Example:

```
🖨 "🖐🏻 🌎"
```

Output:

```
🖐🏻 🌎
```

#### English 🇺🇸

Example:

```
print "Hello World"
```

Output:

```
hello world
```

Example:
